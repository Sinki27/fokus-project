# ⏳ Fokus Project - Cronômetro de Produtividade

O **Fokus** é uma aplicação web de gerenciamento de tempo baseada na técnica Pomodoro. O objetivo é ajudar o usuário a alternar entre períodos de foco total e curtos intervalos de descanso, aumentando a produtividade e o bem-estar.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização moderna e responsiva.
* **JavaScript (ES6+):** Lógica do temporizador, manipulação de eventos e interatividade.

---

## 🛠️ Funcionalidades Principais

* **Cronômetro Regressivo:** Sistema preciso de contagem de tempo para foco e descanso.
* **Alternância de Contextos:** Mudança dinâmica de temas e imagens ao trocar entre "Foco", "Descanso Curto" e "Descanso Longo".
* **Feedback Sonoro:** Alertas de áudio para início, pausa e finalização dos ciclos.
* **Interface Interativa:** Botões de controle intuitivos para iniciar e pausar o contador.

---

## 🧠 Desafios Técnicos Superados

Neste projeto, foquei em aplicar conceitos avançados de **JavaScript**, como:
1.  **Manipulação de DOM:** Alteração de classes e atributos em tempo real para refletir o estado da aplicação.
2.  **Métodos `setInterval` e `clearInterval`:** Controle rigoroso da passagem do tempo e interrupções.
3.  **Tratamento de Áudio:** Implementação de objetos `Audio` para uma experiência de usuário mais imersiva.
